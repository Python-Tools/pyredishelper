watch(self) → ContextManager[Iterable[redisproxy.models.ConsumerRecord]][源代码]¶

同步监听接口.

调用后获得一个管理启动关闭连接并且返回一个可迭代对象的上下文管理器.监听获取消息可以直接使用`for`实现.

示例

c = cast(ConsumerProtocol, kafkac) with c.watch() as g:

for record in g:

print(record.value)

watch(self) → AsyncContextManager[AsyncIterable[redisproxy.models.ConsumerRecord]][源代码]¶

异步监听接口.

调用后获得一个管理启动关闭连接并且返回一个异步可迭代对象的上下文管理器.监听获取消息可以直接使用`async for`实现.
